hurl by defunkt and leah

Hurl makes HTTP requests.

About

Hurl was created for the Rails Rumble 2009 in 48 hours and won Most Complete. Now Hurl is an open source project for your enjoyment.

Code

Clone with Git: 

$ git clone git://github.com/defunkt/hurl

Download in either zip or tar formats.

See our README for installation instructions.
